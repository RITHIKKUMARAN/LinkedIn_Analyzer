## ğŸš€ Quick Start

### Prerequisites

- **Node.js** 18+ installed
- **Python** 3.11+ installed
- **Docker** (Recommended)
- **LinkedIn Account** (Optional, for better scraping)

### Installation

**Option 1: Using Docker (Recommended)**

Clone the repo<br>
git clone https://github.com/RITHIKKUMARAN/LinkedIn_Analyzer.git

Navigate to directory<br>
cd LinkedIn_Analyzer

Start services<br>
docker compose up --build


**Option 2: Manual Setup**

**Backend:**
cd backend<br>
python -m venv venv<br>
source venv/bin/activate  # or .\venv\Scripts\activate<br>
pip install -r requirements.txt<br>
playwright install chromium<br>
uvicorn app.main:app --reload<br>

**Frontend:**
cd frontend<br>
npm install<br>
npm run dev<br>


### Configuration

Create a `.env` file in the `backend` directory:

- Database Config<br>
DATABASE_URL=postgresql+asyncpg://user:pass@localhost:5432/db

- LinkedIn Auth (Optional)<br>
LINKEDIN_USERNAME=your_email<br>
LINKEDIN_PASSWORD=your_password

- AI Config<br>
GEMINI_API_KEY=your_key_here


**Application will launch at:**
- Frontend: `http://localhost:3000` (or 5173)
- Backend: `http://localhost:8000`
- Docs: `http://localhost:8000/docs`
